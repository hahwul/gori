**고리 안에 머무르세요.**

gori(고리 — 영어의 *ring, link, loop*)는 터미널에서 완전히 동작하는, 키보드 중심의 HTTP/HTTPS 인터셉트 프록시이자 웹 해킹 워크벤치입니다. 클라이언트와 타깃 사이의 고리에 자리 잡아 모든 요청을 펜테스트 대상으로 바꾸고, 셸을 벗어나지 않고도 캡처·인터셉트·리피터·퍼징·스캔을 할 수 있게 해 줍니다.
